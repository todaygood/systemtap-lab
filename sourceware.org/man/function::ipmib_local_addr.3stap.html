NAME

function::ipmib_local_addr - Get the local ip address  

SYNOPSIS

    ipmib_local_addr:long(skb:long,SourceIsLocal:long)
 

ARGUMENTS

skb

pointer to a struct sk_buff

SourceIsLocal

flag to indicate whether local operation
 

DESCRIPTION

Returns the local ip address skb.
