NAME

function::isdigit - Checks for a digit  

SYNOPSIS

    isdigit:long(str:string)
 

ARGUMENTS

str

string to check
 

DESCRIPTION

Checks for a digit (0 through 9) as the first character of a string. Returns non-zero if true, and a zero if false.
