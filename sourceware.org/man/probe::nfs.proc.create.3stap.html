NAME

probe::nfs.proc.create - NFS client creating file on server  

SYNOPSIS

nfs.proc.create
 

VALUES

flag

indicates create mode (only for NFSv3 and NFSv4)

filelen

length of file name

fh

file handle of parent dir

prot

transfer protocol

version

NFS version (the function is used for all NFS version)

server_ip

IP address of server

filename

file name
