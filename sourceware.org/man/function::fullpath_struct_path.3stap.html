NAME

function::fullpath_struct_path - get the full path  

SYNOPSIS

    fullpath_struct_path(path:)
 

ARGUMENTS

path

Pointer to "struct path".
 

DESCRIPTION

Returns the full dirent name (full path to the root), like the kernel d_path function.
